

This implementation has the below files. Next to each file, you can find the description of the file.

- hometest.txt			: Problem statement
- betting_client.c		: This source file contains the client implementation
- betting_game_common.c	: This source file contains the implementation common to both the server and the client
- betting_game_common.h	: This header file contains the interfaces and data structures common to both the server and the client
- betting_server.c		: This source file contains the server implementation
- Makefile				: Makefile for the solution, that builds both the server and the client applications
- test_script.sh		: This test script helps in testing the server implementation by calling the client application.
